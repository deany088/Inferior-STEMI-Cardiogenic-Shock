## 📋 Key 2025 Guideline Updates Covered
- **Impella** Class IIa for AMI-CS (DanGer Shock Trial 2024)
- **Intravascular Imaging** (IVUS/OCT) upgraded to Class I
- **Radial Access** Class I for all ACS
- **Complete Revascularization** in STEMI + MVD
- **O₂** only when SpO₂ < 90%
- **Ticagrelor/Prasugrel** preferred over Clopidogrel
